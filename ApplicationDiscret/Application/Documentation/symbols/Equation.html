{void} setFormula(formula)
Creates a formula from the string. The object is completly reset. Every x or y in the formula will be added to the list of variables, and any other string not recognized as a operator will be added to the list of parameters. If the formula is not parsable, the formula, list of parameters and list of variables will be set to undefined.
