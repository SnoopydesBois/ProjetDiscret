{void} setActive(curve, type)
Sets the new active curve for the model. If the type of the curve is provided, a new curve is created with the parameter curve as it's equation. If the type of the curve is not provided, the new active curve is a default curve and the parameter "curve" is the name of this default curve present in the attribute listCurve of the model (i.e the key of the Map).
Parameters:

- The equation of the curve or its name.
